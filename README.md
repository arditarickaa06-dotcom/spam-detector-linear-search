# Deteksi Nomor Telepon Spam

Implementasi algoritma Linear Search untuk mendeteksi nomor telepon spam berdasarkan database blacklist.

## Fitur
- Deteksi nomor telepon spam
- Implementasi algoritma Linear Search
- Menampilkan status SPAM atau BUKAN SPAM
- Menampilkan jumlah perbandingan
- Menampilkan waktu eksekusi
- Menampilkan posisi data ditemukan

## Teknologi
- HTML
- CSS
- JavaScript
- Bootstrap 5

## Cara Menjalankan
1. Download repository.
2. Buka folder menggunakan Visual Studio Code.
3. Install extension Live Server.
4. Jalankan file `index.html`.
